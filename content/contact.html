Thanks in advance for contacting me.

In order for me to answer you as soon as possible, here are the best communication methods:

  • If you have a question or a suggestion related to an article, I invite you to add it as a comment (at the end of the corresponding article) so other readers can benefit from the discussion
  • For mistakes or bugs, you can inform me about them by raising an issue on GitHub. Note that the code for all articles and pages of this blog can be found on GitHub
  • For all other requests, please use the contact form below
  • Note that if you need to send a file, first fill in the contact form below to which I will reply and from there you will be able to send me your file

For a quick answer to your question, make sure to check the FAQ first.
