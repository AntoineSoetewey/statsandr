## 1. Purpose

This Code of Conduct exists to help ensure that this site and its associated community — including interactions on GitHub — remain welcoming, respectful, and constructive for everyone. It applies to comments on the website, GitHub discussions and issues, pull requests, contributor submissions, and any communication related to this project.

## 2. Our Values

We aim to foster a community built on:

- **Respect and kindness** — Treat others with empathy, patience, and courtesy.
- **Inclusion and openness** — Welcome and respect people of all identities, backgrounds, and perspectives.
- **Collaboration and learning** — Encourage curiosity, knowledge sharing, and constructive dialogue.
- **Integrity and honesty** — Attribute ideas correctly, communicate clearly, and engage in good faith.

## 3. Expected Behavior

By participating in this project or community, you agree to:

- Communicate respectfully and thoughtfully in all spaces (website comments, GitHub issues, pull requests, or emails).
- Use inclusive and non-discriminatory language.
- Focus discussion on ideas rather than individuals.
- Offer feedback constructively and accept feedback graciously.
- Credit others' work appropriately and avoid plagiarism.
- Follow repository contribution guidelines when submitting work through GitHub.
- Help maintain a safe and supportive environment for all participants.

## 4. Unacceptable Behavior

The following are not permitted anywhere in this community:

- Harassment, hate speech, or discriminatory remarks based on (including but not limited to): race, ethnicity, nationality, religion, gender identity, sexual orientation, disability, age, body size, political identity, or socioeconomic status.
- Personal attacks, insults, intimidation, or threats.
- Posting or promoting spam, harmful content, misinformation, or malicious code.
- Dismissive, hostile, or mocking responses to questions or contributions.
- Plagiarism or misappropriation of intellectual work.
- Repeated disruption of discussions or contributions.
- Any behavior that makes another community member feel unsafe or unwelcome.

## 5. Reporting and Enforcement

If you observe or experience behavior that violates this Code of Conduct, you may report it using the contact options on the website or directly through GitHub.

All reports will be:

- Taken seriously,
- Handled confidentially whenever possible,
- Reviewed in a fair and impartial manner.

Consequences for violations may include (depending on severity):

- Removal or editing of content,
- Closing or rejecting pull requests or issues,
- Temporary or permanent banning from participation,
- Revocation of contributor or collaboration privileges.

## 6. Scope

This Code of Conduct applies to all project spaces, including:

- This website and its comment sections  
- GitHub issues, discussions, and pull requests  
- Direct communications related to the project (email or otherwise)  

This Code does **not** prohibit disagreement or critical discussion — provided it remains respectful, evidence-based, and aligned with the values above.

## 7. A Living Document

This Code of Conduct may evolve as the community grows. Suggestions for improvements are welcome and can be proposed via a GitHub issue or pull request.

Thank you for contributing to a respectful, thoughtful, and inclusive community.
