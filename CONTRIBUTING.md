## Contributing via Pull Requests

If you would like to submit a fix or enhancement directly:

1. Fork the repository  
2. Create a new branch  
3. Make your changes  
4. Submit a pull request  

Please ensure that:

- Your contribution is clearly documented
- Code snippets run without errors
- Changes maintain the tone and educational intent of the site

Pull requests may not be merged immediately and may require discussion or revision.
