# Support

Thank you for your interest in **Stats and R**.

If you have:

- A **question about content** → comment on the article or open a GitHub issue.
- A **correction or improvement suggestion** → open a GitHub issue or pull request.
- A **guest post proposal** → use the contribution form on the site: https://statsandr.com/contribute/
- A **general inquiry or message** → https://statsandr.com/contact/
